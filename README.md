c. Página 2columnas.css

Esta forma se aplicó solo a las paginas web almacenamiento, fuente de poder, procesador y ram
(almacenamiento.html, fuente-poder.html, procesador.html, ram.html).

Imágenes de fondo: Una gran parte de páginas web utilizan imágenes para presentar una pagina web, eso
visualmente es atractivo, ya que generaliza el tema que se está tratando.

El comando de <background-image> asigna una imagen de fondo la cual puede redimensionarse o desplazarse
con el scroll del ratón, en este caso la imagen se redimensiona al panel de la página web.

d. Página 3columnas.css

Esta forma se aplicó solo a las páginas web perifericos, placa madre y tarjeta de video (perifericos.html, placa-
madre.html, tarjeta-video.html).

Tablas: Aplicando el requerimiento que se necesita aplicar una taba en la página web, está alineada al <aside>
donde está la publicidad. Para el comando <border-collapse> se utiliza para fusionar los bordes. Ésto tiene una
gran influencia sobre la presentación y el estilo de las celdas de tabla.

Repetir la imagen: Se aplico el comando de no repetir la imagen ya que dañaba el diseño de las páginas de 3
columnas.

e. Página cuenta.css

Esta forma se aplicó solo a la página cuenta (cuenta.html)
